# Runtime Regression Checklist

## 2인 게임 포기

1. A/B로 새 2인 게임을 시작한다.
2. A가 `게임 포기 및 나가기`를 누른다.
3. A는 FORFEITED, B는 WINNER로 게임이 종료된다.
4. A/B 모두 로비로 이동한다.
5. 종료된 방이 로비 목록에서 즉시 사라진다.
6. A가 새 방을 만든다.
7. B가 새 방에 입장한다.
8. A/B READY 후 새 게임이 시작된다.

## 대기방 나가기

1. 한 계정이 새 대기방을 만든다.
2. `방 나가기`를 누른다.
3. 로비로 이동한다.
4. 해당 방은 목록에서 사라진다.
5. 같은 계정이 곧바로 새 방을 만들 수 있다.

## 재접속

- 포기 후 새로고침·재로그인해도 종료 게임으로 복귀하지 않는다.
- 단순 로그아웃만 한 진행 중 게임은 기존 정책대로 복귀 가능하다.
